Primeiros passos com HTML && CSS



  • O HTML (HyperText Markup Language),
    • como o nome sugere, é uma linguagem de marcação de texto
  • usada para estruturar e formatar conteúdos em uma página web

Por meio das chamadas TAGS somos capazes de capturar e reestruturar um novo padrão de texto

Exemplos de TAGS HTML:
  • <p> Marca um parágrafo com quebra de linha
  • <a href="https://www.exemplo.com"> Link para um site externo </a>
  • <img> Adiciona uma imagem ao site
  • <h1> Título principal


Estilização em cascata

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a apresentação e o layout de elementos HTML em páginas da web. Ele permite separar o conteúdo (HTML) da sua apresentação (estilo), tornando mais fácil e eficiente o controle sobre a aparência visual de um site. Ele pode ser usado para criar uma ampla variedade de estilos, desde formatação simples até layouts complexos e responsivos. Além disso, o CSS oferece a capacidade de animar elementos e adicionar efeitos especiais para tornar a experiência do usuário mais atraente. Com suas poderosas funcionalidades, o CSS é uma ferramenta essencial para qualquer desenvolvedor web.
